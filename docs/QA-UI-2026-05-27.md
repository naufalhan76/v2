## Recommended Fix Priority

### Immediate (CRIT — before next deploy)

1. `technician/layout.tsx` — Remove `userScalable: false` from viewport config.
2. `dashboard/layout.tsx` — Replace `fixed inset-0 overflow-hidden` root wrapper with a flex-column layout that does not trap scroll on iOS.
3. `operasional/create-order/page.tsx:997` — Add `sm:grid-cols-2` breakpoint to schedule grid (make it single column on mobile).

### Sprint 1 (HIGH — UX/A11y blockers)

1. Add `aria-label` to all icon-only buttons across customer, technician, reminder-rules, and invoice detail pages.
2. Translate `operasional/create-order` and `manajemen/teknisi` pages to Indonesian.
3. Add `fallback={<OrdersPageSkeleton />}` to `orders/page.tsx` Suspense boundary.
4. Fix hardcoded `bg-blue-50`/`text-blue-900` and `text-gray-*` colors to use CSS variables for dark mode.
5. Fix `router.back()` in invoice detail to fall back to `/dashboard/keuangan/invoices`.

### Sprint 2 (MED — polish)

1. Add breadcrumbs to deep pages (orders/new, invoices/[id], customer/[id]).
2. Update quick action links on dashboard to point to new `/dashboard/orders` routes.
3. Add `locale={id}` to all Calendar components.
4. Debounce technician search input.
5. Fix `window.location.href` navigation to use `router.push`.
6. Fix empty state on dashboard recent orders to use `<EmptyState>` component.
